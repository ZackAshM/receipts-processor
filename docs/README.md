# docs

Purpose:
- Product-management and governance documentation for this project.

Contents:
- `PRD.md`: product goals and scope.
- `ARCHITECTURE.md`: system design and flow.
- `DECISIONS.md`: decision log with requested changes.
- `CHANGELOG.md`: chronological change history.
- `OPERATIONS.md`: runbook and operating model.
- `INIT_PROMPT.txt`: static initial prompt artifact.
- `CODEX_LOG.md`: append-only chat log with redaction safeguards.
