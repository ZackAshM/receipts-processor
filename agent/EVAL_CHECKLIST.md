# Evaluation Checklist

## Functional

- [ ] Supported files are discovered correctly.
- [ ] Output file matches selected model schema.
- [ ] CSV and XLSX export modes both work.

## Quality

- [ ] Required fields validation is active.
- [ ] Confidence score is computed and stored.
- [ ] Exception routing is available.

## Governance

- [ ] Human approval process is documented and followed.
- [ ] Risk register covers top operational risks.
- [ ] Prompt and change logs are current.
